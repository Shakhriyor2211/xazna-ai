## Endpoint

`https://ai.xazna.uz/api/v1/llm/token/session/generate`

`wss://ai.xazna.uz/ws/v1/llm/user/message/<session-id>`

## Authentication

Include your token in each request, either in the request header as X-Access-Token or as a query parameter:

```
X-Access-Token: <your-token>
```

```
token=<your-token>
```

## Session request

| Field   | Type   | Description                           |
| ------- | ------ | ------------------------------------- |
| content | string | Initial message to start conversation |
| mdl     | string | Conversation model `Base`             |

## Session message

| Field   | Type   | Description                             |
| ------- | ------ | --------------------------------------- |
| content | string | User message                            |
| mdl     | string | Conversation model `Base`               |
| action  | string | Conversation action `message`, `cancel` |

```javascript
import fetch from "node-fetch";
import fs from "fs";

async function create_session() {
  const formData = new FormData();
  formData.append("audio", fs.createReadStream("example.mp3"));
  formData.append("mdl", "Base");

  const response = await fetch(
    "https://ai.xazna.uz/api/v1/llm/token/session/generate",
    {
      method: "POST",
      headers: {
        "X-Access-Token": "<your-token>",
      },
      body: JSON.stringify({
        content: "Salom! Bu namuna matn.",
        mdl: "Base",
      }),
    }
  );

  const data = await response.json();
  console.log("Slug:", data.slug);
}

create_session();
```

```javascript
const WebSocket = require("ws");

const ws = new WebSocket(
  "wss://ai.xazna.uz/ws/v1/llm/token/message/<session-id>",
  {
    headers: {
      "X-Access-Token": "<your-token>",
```

```python
import websocket
import json

ws = websocket.WebSocket()

ws.connect(
  "wss://ai.xazna.uz/ws/v1/llm/token/message/<session-id>",
  header={"X-Access-Token": "<your-token>"}
)

ws.send(json.dumps({
    "content": "Salom! Bu namuna matn.",
    "action": "message",
    "mdl": "Base"
}))

while True:
    data = ws.recv()
    print("Received:", data)
```

## Response

### Session

- **slug**: _string_  
  Session id to establish websocket connection.

### Conversation

- **status**: _number_  
  Message status code
- **type**: _string_  
  Message type `stream`, `error`
