Xazna AI enforces rates for credits to ensure fair usage and maintain service stability for all users. The following rates are in effect for chatbot, text to speech (TTS) and speech to text (STT) endpoints and rates vary depending on the active subscription plan.

## Handling Rate Errors

### Chatbot

| Status | Message                                 |
| ------ | --------------------------------------- |
| 403    | Message limit reached, open new session |
| 403    | Not enough funds                        |
| 403    | Not enough funds                        |
| 429    | Request limit exceeded                  |

### Text to speech (TTS)

| Status | Message                |
| ------ | ---------------------- |
| 403    | Not enough credits     |
| 403    | Not enough founds      |
| 429    | Request limit exceeded |

### Speech to text (STT)

| Status | Message                |
| ------ | ---------------------- |
| 403    | Not enough credits     |
| 403    | Not enough founds      |
| 429    | Request limit exceeded |
