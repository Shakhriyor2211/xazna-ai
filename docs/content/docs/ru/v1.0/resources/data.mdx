В Xazna AI мы стремимся защищать вашу конфиденциальность и предоставлять вам контроль над вашими данными. Эта страница объясняет, как мы обрабатываем данные, которые вы предоставляете при использовании наших сервисов чат-бота, преобразования текста в речь (TTS) и речи в текст (STT), а также ваши возможности по их управлению.

## Сбор данных

При использовании сервисов Xazna AI мы собираем следующие данные для предоставления и поддержки наших услуг:

- **Текстовые данные**: текст, который вы отправляете для преобразования в речь (TTS).
- **Аудио данные**: аудиофайлы, которые вы загружаете для транскрипции речи в текст (STT).
- **Данные об использовании**: метаданные, такие как временные метки, IP-адреса и детали запросов, чтобы обеспечить надежность сервиса и точность выставления счетов.

Эти данные обрабатываются исключительно для предоставления запрошенных вами услуг.

## Использование данных

Мы используем ваши данные для следующих целей:

- **Предоставление услуг**: для преобразования вашего текста в речь или транскрипции аудио в текст.
- **Аналитика**: для отслеживания шаблонов использования и поддержания производительности сервиса, с агрегированными данными, которые не идентифицируют отдельных пользователей.
- **Выставление счетов**: для расчёта расходов на операции с чат-ботом, TTS и STT.

## Безопасность

Мы применяем стандарты отраслевой безопасности для защиты ваших данных, включая:

- Шифрование данных при передаче и хранении.
- Контроль доступа для ограничения данных только уполномоченному персоналу.

## Политика

Xazna AI соблюдает применимые законы о защите данных, включая правила обработки персональных данных в регионах, где предоставляются наши услуги. Для конкретных запросов свяжитесь с нашей юридической командой через [ai.xazna.uz](https://ai.xazna.uz).
