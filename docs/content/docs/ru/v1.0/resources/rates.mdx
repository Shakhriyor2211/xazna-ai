Xazna AI применяет тарифные ограничения на использование кредитов, чтобы обеспечить справедливое использование и стабильную работу сервиса для всех пользователей. Указанные ниже тарифы действуют для конечных точек чат-бота, преобразования текста в речь (TTS) и речи в текст (STT) и зависят от активного тарифного плана.

## Обработка ошибок, связанных с лимитами

### Чат-бот

| Статус | Сообщение                                         |
| ------ | ------------------------------------------------- |
| 403    | Превышен лимит сообщений, откройте новую сессию. |
| 403    | Недостаточно кредитов                             |
| 403    | Недостаточно средств                              |
| 429    | Превышен лимит запросов                           |

### Преобразование текста в речь (TTS)

| Статус | Сообщение               |
| ------ | ----------------------- |
| 403    | Недостаточно кредитов   |
| 403    | Недостаточно средств    |
| 429    | Превышен лимит запросов |

### Преобразование речи в текст (STT)

| Статус | Сообщение               |
| ------ | ----------------------- |
| 403    | Недостаточно кредитов   |
| 403    | Недостаточно средств    |
| 429    | Превышен лимит запросов |
