## Endpoint

`https://ai.xazna.uz/api/v1/stt/token/generate`

## Аутентификация

Включите ваш токен в каждый запрос — либо в заголовке X-Access-Token, либо как параметр запроса:

```
X-Access-Token: <your-token>
```

```
token=<your-token>
```

## Тело запроса

| Поле  | Тип    | Описание                                |
| ----- | ------ | --------------------------------------- |
| audio | file   | Аудиофайл для транскрипции `mp3`, `wav` |
| mdl   | string | Модель транскрипции `Base`              |

## Пример использования

<Tabs items={['curl', 'javascript', 'python']} defaultValue="curl">

```bash
curl -X POST "https://ai.xazna.uz/api/v1/stt/token/generate" \
  -H "Content-Type: multipart/form-data" \
  -H "X-Access-Token: <your-token>" \
  -F "audio=@example.mp3" \
  -F "mdl=Base"
```

<Tab value="javascript">

```javascript
import fetch from "node-fetch";
import fs from "fs";

async function stt() {
  const formData = new FormData();
  formData.append("audio", fs.createReadStream("example.mp3"));
  formData.append("mdl", "Base");

  const response = await fetch(
    "https://ai.xazna.uz/api/v1/stt/token/generate",
    {
      method: "POST",
      headers: {
        "X-Access-Token": "<your-token>",
      },
      body: formData,
    }
  );

  const data = await response.json();
  console.log("Transcription:", data.transcript);
}

stt();
```
