## Endpoint

`https://ai.xazna.uz/api/v1/llm/token/session/generate`

`wss://ai.xazna.uz/ws/v1/llm/user/message/<session-id>`

## Аутентификация

Включите ваш токен в каждый запрос — либо в заголовке запроса X-Access-Token, либо в качестве параметра запроса:

```
X-Access-Token: <your-token>
```

```
token=<your-token>
```

## Запрос создания сессии

| Поле    | Тип    | Описание                               |
| ------- | ------ | -------------------------------------- |
| content | string | Начальное сообщение для начала диалога |
| mdl     | string | Модель диалога `Base`                  |

## Сообщение сессии

| Поле    | Тип    | Описание                             |
| ------- | ------ | ------------------------------------ |
| content | string | Сообщение пользователя               |
| mdl     | string | Модель диалога `Base`                |
| action  | string | Действие диалога `message`, `cancel` |

import fetch from "node-fetch";
import fs from "fs";

async function create_session() {
  const formData = new FormData();
  formData.append("audio", fs.createReadStream("example.mp3"));
  formData.append("mdl", "Base");

  const response = await fetch(
    "https://ai.xazna.uz/api/v1/llm/token/session/generate",
    {
      method: "POST",
      headers: {
        "X-Access-Token": "<your-token>",
      },
      body: JSON.stringify({
        content: "Salom! Bu namuna matn.",
        mdl: "Base",
      }),
    }
  );

  const data = await response.json();
  console.log("Slug:", data.slug);
}

create_session();
```

import websocket
import json

ws = websocket.WebSocket()

ws.connect(
  "wss://ai.xazna.uz/ws/v1/llm/token/message/<session-id>",
  header={"X-Access-Token": "<your-token>"}
)

ws.send(json.dumps({
    "content": "Salom! Bu namuna matn.",
    "action": "message",
    "mdl": "Base"
}))

while True:
    data = ws.recv()
    print("Received:", data)
```

## Ответ

### Сессия

- **slug**: _string_  
  Идентификатор сессии для установления WebSocket-соединения.

### Диалог

- **status**: _number_  
  Код статуса сообщения
- **type**: _string_  
  Тип сообщения `stream`, `error`
