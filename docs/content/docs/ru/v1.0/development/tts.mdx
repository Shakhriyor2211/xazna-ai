## Endpoint

`https://ai.xazna.uz/api/v1/tts/token/generate`

## Аутентификация

Включите ваш токен в каждый запрос — либо в заголовке X-Access-Token, либо как параметр запроса:

```
X-Access-Token: <your-token>
```

```
token=<your-token>
```

## Тело запроса

| Поле    | Тип    | Описание                                          |
| ------- | ------ | ------------------------------------------------- |
| text    | string | Текст для синтеза речи (макс. 5000 символов)      |
| mdl     | string | Голосовая модель `SurAio`                         |
| format  | string | Формат выхода: `mp3`, `wav`, `aac`, `ogg`, `flac` |
| emotion | string | Эмоция `Neural`                                   |

## Пример использования

<Tabs items={['curl', 'javascript', 'python']}>
<Tab value="curl">

```bash
curl -X POST "https://ai.xazna.uz/api/v1/tts/token/generate" \
  -H "Content-Type: application/json" \
  -H "X-Access-Token: <your-token>" \
  -d '{
    "text": "Salom! Bu namuna matn.",
    "mdl": "SurAio",
    "format": "mp3",
    "emotion": "Neural"
  }' \
  --output output.mp3
```

</Tab>
<Tab value="javascript">

```javascript
import fetch from "node-fetch";
import fs from "fs";

async function tts() {
  const response = await fetch(
    "https://ai.xazna.uz/api/v1/tts/token/generate",
    {
      method: "POST",
      headers: {
        "Content-Type": "application/json",
        "X-Access-Token": "<your-token>",
      },
      body: JSON.stringify({
        text: "Salom! Bu namuna matn.",
        mdl: "SurAio",
        format: "mp3",
        emotion: "Neural",
      }),
    }
  );

  const buffer = await response.arrayBuffer();
  fs.writeFileSync("output.mp3", Buffer.from(buffer));
}

tts();
```

<Tab value="python">

```python
import requests

url = "https://ai.xazna.uz/api/v1/tts/token/generate"
headers = {
    "Content-Type": "application/json",
    "X-Access-Token": "<your-token>"
}
payload = {
    "text": "Salom! Bu namuna matn.",
    "mdl": "SurAio",
    "format": "mp3",
    "emotion": "Neural"
}

response = requests.post(url, json=payload)
with open("output.mp3", "wb") as f:
    f.write(response.content)
```
