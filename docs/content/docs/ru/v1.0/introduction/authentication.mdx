---

API Xazna AI использует аутентификацию на основе токенов. Действительный API-токен должен быть включён во все запросы к конечным точкам преобразования текста в речь (TTS) и речи в текст (STT).

## Получение токена

1. Перейдите на сайт [ai.xazna.uz](https://ai.xazna.uz).
2. Войдите, используя данные своей учётной записи Xazna AI.
3. Откройте раздел API, чтобы создать или получить ваш API-токен.
4. Храните токен в безопасности и не передавайте его третьим лицам.

<Callout type="warning">
  Не публикуйте свой API-токен. Если он был скомпрометирован, создайте новый
  токен на [ai.xazna.uz](https://ai.xazna.uz).
</Callout>

## Обработка ошибок аутентификации

| Статус | Сообщение                   | Код              |
| ------ | --------------------------- | ---------------- |
| 401    | Токен не предоставлен       | token_required   |
| 403    | Токен заблокирован          | token_blocked    |
| 403    | Токен неактивен             | token_inactive   |
| 403    | Учётная запись заблокирован | account_blocked  |
| 403    | Учётная запись неактивна    | account_inactive |

<Callout type="info">
  Обратитесь в службу поддержки через [ai.xazna.uz](https://ai.xazna.uz), если у
  вас возникли проблемы с генерацией токена.
</Callout>
