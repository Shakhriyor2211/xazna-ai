Добро пожаловать в документацию Xazna AI! Это руководство является отправной точкой для знакомства с возможностями платформы Xazna AI, которая предоставляет современные сервисы чат-ботов, преобразования текста в речь (TTS) и речи в текст (STT). Независимо от того, являетесь ли вы разработчиком, планирующим интеграцию наших API, или пользователем, желающим лучше понять наши возможности, эта документация поможет вам начать работу.

Начните работу с Xazna AI, изучив следующие разделы:

- [Аутентификация](./authentication) — для защиты доступа к API.
- [Модели](./models) — чтобы узнать о доступных голосовых и языковых возможностях.

<Callout type="info">
  Посетите [ai.xazna.uz](https://ai.xazna.uz), чтобы сгенерировать API-токен.
</Callout>
