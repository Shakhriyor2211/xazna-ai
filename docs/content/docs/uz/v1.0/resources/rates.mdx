Xazna AI barcha foydalanuvchilar uchun adolatli foydalanishni ta'minlash va xizmat barqarorligini saqlash maqsadida kreditlar bo'yicha tarif cheklovlarini qo'llaydi. Quyidagi tariflar chatbot, matndan nutqqa (TTS) va nutqdan matnga (STT) endpointlari uchun amal qiladi hamda faol obuna rejasiga qarab farqlanadi.

## Tarif xatolarini qayta ishlash

### Chatbot

| Status | Xabar                                                   |
| ------ | ------------------------------------------------------- |
| 403    | Xabarlar chegarasidan oshib ketdi, yangi sessiya oching |
| 403    | Kreditlar yetarli emas                                  |
| 403    | Mablag' yetarli emas                                    |
| 429    | So'rovlar chegarasidan oshib ketdi.                     |

### Matndan nutqqa (TTS)

| Status | Xabar                               |
| ------ | ----------------------------------- |
| 403    | Kreditlar yetarli emas              |
| 403    | Mablag' yetarli emas                |
| 429    | So'rovlar chegarasidan oshib ketdi. |

### Nutqdan matnga (STT)

| Status | Xabar                               |
| ------ | ----------------------------------- |
| 403    | Kreditlar yetarli emas              |
| 403    | Mablag' yetarli emas                |
| 429    | So'rovlar chegarasidan oshib ketdi. |
