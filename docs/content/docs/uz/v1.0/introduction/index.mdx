Xazna AI hujjatlariga xush kelibsiz! Ushbu qo'llanma sizni chatbot, matndan nutqqa (TTS) va nutqdan matnga (STT) xizmatlarini taqdim etuvchi Xazna AI platformasining imkoniyatlari bilan tanishtirish uchun boshlang'ich nuqta bo'lib xizmat qiladi. Siz API'larimizni integratsiya qilmoqchi bo'lgan dasturchi yoki xizmatlarimizni yaxshiroq tushunishni istagan foydalanuvchi bo'lishingizdan qat'i nazar, ushbu hujjatlar sizga boshlashda yordam beradi.

Xazna AI bilan ishlashni quyidagi bo'limlarni o'rganishdan boshlang:

- [Autentifikatsiya](./authentication) — API'ga xavfsiz kirishni ta'minlash uchun.
- [Modellar](./models) — mavjud ovoz va til imkoniyatlari bilan tanishish uchun.

<Callout type="info">
  API tokeningizni yaratish uchun [ai.xazna.uz](https://ai.xazna.uz) saytiga tashrif buyuring.
</Callout>

## Ishlab chiqish

Dasturchilar uchun quyidagi bo'limlar foydali bo'ladi:

- Node.js, Python va boshqa texnologiyalar bilan Xazna AI'ni integratsiya qilish uchun API'dan foydalanish.

## Resurslar

Qo'shimcha ma'lumotlar quyidagi bo'limlarda mavjud:

- [Ma'lumotlaringiz](../v1.0/resources/your-data) — ma'lumotlarni qayta ishlash siyosati.
- [Foydalanish cheklovlari](../v1.0/resources/rate-limits) — foydalanish qoidalari va cheklovlari.
